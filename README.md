Memory Allocation Simulator



A web-based simulator, built with React and Tailwind CSS, that visually demonstrates and compares three core memory allocation algorithms: First Fit, Best Fit, and Worst Fit.



This project was created as part of a university course to understand the fundamental concepts of operating systems and memory management.



**‚ú® Features**



Three Strategies: Implements and compares First Fit, Best Fit, and Worst Fit side-by-side.



Dynamic Configuration: Allows you to add and remove initial memory blocks (partitions) of any size.



Process Queue: Create a queue of processes with custom sizes to be allocated.



Visual Demonstration: Visually represents memory as colored blocks (blue for allocated, green for free), making it easy to see fragmentation.



Interactive Deallocation: Click "Free" to deallocate any process and watch the memory blocks automatically coalesce (merge).



Fully Responsive: Designed to work on both desktop and mobile browsers.
